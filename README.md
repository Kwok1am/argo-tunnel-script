# CloudFlare Argo Tunnel Onekey Script

## Usage

Download this script:
```shell
wget -N --no-check-certificate https://raw.githubusercontents.com/yuuki410/argo-tunnel-script/master/argo.sh
```

Run this script:
```shell
bash argo.sh
```
